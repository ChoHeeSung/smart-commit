# 대화형 저장소 선택 기능 추가

## 지시 요약
스캔 후 표시되는 레포지토리 리스트에서 사용자가 처리할 레포만 체크박스로 선택할 수 있는 기능 요청.

## 작업 내용

### 문제
기존에는 dirty 상태인 레포지토리를 모두 자동 순회하며 처리했다. 다수의 레포를 관리할 때 특정 레포만 골라서 커밋하고 싶은 경우가 빈번한데, 이를 위한 선택 UI가 없었다.

### 해결
terminal-kit 기반의 대화형 체크박스 UI를 구현하여, 스캔 결과 테이블 출력 직후 사용자가 처리 대상 레포를 선택할 수 있게 했다.

### 비유: 편의점 택배 접수
> 택배 접수대에 물건 10개를 올려놨을 때, 기존에는 직원이 "전부 보내겠습니다"하고 일괄 발송했다.
> 이제는 물건 목록을 보여주고 "어떤 것을 보낼까요?" 하고 체크리스트를 건네준다.
> 고객은 보낼 것만 ✓ 표시하고, 나머지는 다음에 보내면 된다.

## 변경 파일

| 파일 | 변경 내용 |
|------|----------|
| `src/i18n.ts` | `selectRepos`, `selectReposHint`, `noReposSelected` 메시지 한/영 추가 |
| `src/ui.ts` | `selectRepos()` 메서드 — 체크박스 UI (↑↓ 이동, Space 토글, a 전체, Enter 확정, ESC 취소) |
| `src/index.ts` | `showRepoTable()` 후 `selectRepos()` 호출, 선택된 레포만 처리하도록 필터링 |

## 핵심 동작
- dirty 레포가 0개 → 빈 배열 반환
- dirty 레포가 1개 → 선택 UI 없이 바로 반환 (자동 선택)
- dirty 레포가 2개 이상 → 체크박스 UI 표시 (초기값: 전부 체크)
- headless 모드(`--no-interactive`) → 선택 UI 건너뜀 (전체 처리)
